== Description ==

Zen Blocks is a powerful WordPress plugin that simplifies the creation of custom Gutenberg blocks using HTML/PHP templates. No JavaScript knowledge required!

**Key Features:**

* Create blocks using simple HTML templates
* Support for dynamic PHP templates
* Various editable field types including text, images, buttons, and more
* Repeater fields for creating dynamic content
* InnerBlocks support for nested Gutenberg blocks
* Simple JSON configuration for block settings

**Template Types:**

1. **Static HTML Templates**
   Create blocks with a fixed structure using HTML and special attributes to define editable areas.

2. **Dynamic PHP Templates**
   Build advanced blocks with PHP logic, conditional rendering, and dynamic content.

== Installation ==

1. Upload the `zen-blocks` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Start creating custom blocks using HTML/PHP templates

== Frequently Asked Questions ==

= Do I need to know JavaScript to create blocks? =

No! Zen Blocks allows you to create custom Gutenberg blocks using only HTML and PHP, with no JavaScript knowledge required.

= How do I create a new block? =

1. Create an HTML or PHP template file
2. Add special attributes like `zen-edit` to define editable areas
3. Create a JSON configuration file alongside your template
4. Your block will automatically appear in the Gutenberg editor

= Can I create dynamic blocks? =

Yes! Zen Blocks supports dynamic PHP templates with conditional logic, loops, and database queries.
